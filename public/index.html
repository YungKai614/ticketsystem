Ablauf & Ihre Rechte.

Transparent von Anfang bis Ende — so läuft eine Anfrage ab, und das können Sie von uns erwarten.

So läuft es ab
1. Anfrage einreichen
Online über das Portal oder Formular — mit Beschreibung des Problems und Ihren Kontaktdaten.
2. Bestätigung & Ticket-ID
Sie erhalten sofort eine Ticket-ID zur Nachverfolgung. SLA-Kunden bekommen zusätzlich eine E-Mail-Bestätigung.
3. Bearbeitung durch unser Team
Ihr Ticket wird dem zuständigen Techniker zugewiesen und innerhalb der vereinbarten Reaktionszeit bearbeitet.
4. Rückmeldung & Lösung
Wir melden uns mit einer Lösung oder einem Zwischenstand. Bei Bedarf wird das Ticket an Spezialisten weitergeleitet.
5. Abschluss & Feedback
Nach der Lösung können Sie Feedback geben. Externe Anfragen erhalten eine detaillierte Rechnung.
Ihre Rechte
Sie erhalten vorab eine Einschätzung des Aufwands. Ohne Ihre Zustimmung wird keine kostenpflichtige Arbeit begonnen. Abgerechnet wird nur tatsächlich geleistete Arbeit. Material wird zum Einkaufspreis plus einem transparenten Aufschlag verrechnet.
Die vereinbarte Reaktionszeit ist garantiert. Bei Nichteinhaltung erhalten Sie eine Gutschrift von CHF 50 pro Vorfall. Die Verfügbarkeitsgarantie gilt pro Kalendermonat. Alle Garantien sind im SLA-Vertrag schriftlich festgehalten.
SLA-Abonnements haben eine Mindestlaufzeit von 12 Monaten. Kündigung schriftlich 3 Monate vor Vertragsablauf. Ohne Kündigung verlängert sich der Vertrag automatisch um 12 Monate. Upgrade jederzeit möglich, Downgrade mit 4 Wochen Vorankündigung.
Alle Anfragen und Kundendaten werden vertraulich behandelt. Keine Weitergabe an Dritte. Daten werden nur für die Bearbeitung Ihrer Anfrage verwendet und nach Abschluss gemäss Datenschutzgesetz archiviert.
Bei externen Anfragen zahlen Sie nur für tatsächlich geleistete Arbeit — auch wenn das Problem nicht vollständig gelöst werden konnte. Bei SLA-Kunden eskalieren wir das Problem an den Hersteller (3rd Level) — auf unsere Kosten.

Hilfe & Anleitung

Schnellreferenz für den Supporter-Alltag. Klicke auf ein Thema um die Anleitung aufzuklappen.

1. Seitenleiste: Inbox klicken
2. Dropdown «Inbox für» — Deinen Namen wählen
3. Du siehst nur Tickets die Dir zugewiesen sind
4. Tab «Offen» zeigt neue Tickets die auf Bearbeitung warten
1. Ticket in der Inbox anklicken
2. Button «Bearbeiten» klicken (oben rechts)
3. «Zugewiesen an» — Deinen Namen wählen
4. Status auf «In Bearbeitung» setzen
5. «Änderungen speichern» klicken

Blauer Balken links = Kunde wartet auf Antwort! Zuerst diese bearbeiten.
1. Ticket öffnen
2. Button «Antworten» unten links
3. Antworttext eingeben
4. «Senden» klicken

Der blaue Balken verschwindet nach dem Antworten automatisch.
Wann eskalieren?
— Netzwerk / VPN / Firewall → 2nd Level Network Engineer
— Server / AD / Backup → 2nd Level System Engineer
— Hersteller-Fehler → 3rd Level Extern
— Sicherheitsvorfall → sofort Nils Müller informieren

So eskalieren:
1. Ticket öffnen → «Bearbeiten»
2. Level ändern: 1st → 2nd oder 3rd
3. «Zugewiesen an»: Kollegen auswählen
4. «Antworten» → kurze Begründung schreiben (WARUM?)
5. «Änderungen speichern»
1. Seitenleiste: Externe Tickets
2. Ticket öffnen → «Bearbeiten»
3. Button «Verrechnen» erscheint unten
4. Beschreibung + Menge + CHF/Einheit eingeben
5. «+ Hinzufügen» → Total berechnet sich automatisch
6. «Verrechnung speichern»
7. «Gelöst» klicken

Tipp: Ohne Verrechnung erscheint eine Warnung beim Abschliessen.
1. Seitenleiste: Alle Tickets
2. Filter-Dropdowns auswählen (Status, SLA, Priorität)
3. «+ Filter speichern» klicken
4. Namen eingeben (z. B. «Kritisch und Offen»)
5. Filter erscheint als Schaltfläche mit Live-Anzahl

Gespeicherte Filter sind nur für Dich sichtbar.
